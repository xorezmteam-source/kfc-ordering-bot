# 🍗 KFC Smart Ordering Bot — Voice & Chat AI

A conversational AI ordering system for KFC built with Google Dialogflow, supporting both voice and chat interfaces with full menu customization.

## ✨ Features

- 🛒 **Full Menu Ordering** — Burgers, drinks, sides with size selection
- 🥗 **Dietary Filters** — Allergy and ingredient preferences
- 🚗 **Delivery / Pickup** — Customer selects preferred method
- 📍 **Branch Locator** — Find nearest KFC branch
- ✅ **Order Confirmation** — Smart yes/no followup flow

## 🛠 Tech Stack

- **NLP Engine:** Google Dialogflow (English)
- **Interface:** Voice (Alexa-compatible) + Chat
- **Entities:** Menu items, sauces, drinks, branch locations, allergens

## 📁 Intent Structure

```
intents/
├── Default Welcome Intent   # Greeting
├── KFC Order                # Main ordering flow
├── KFC Order - yes          # Confirmation
├── KFC Order - no           # Cancel/modify
├── Branch Info              # Location lookup
└── Goodbye                  # Session end
```

## 🎯 Use Cases

This bot can be adapted for any restaurant chain needing:
- Automated order taking (reduce staff costs)
- 24/7 customer service
- Multi-platform support (Telegram, Web, Alexa)
